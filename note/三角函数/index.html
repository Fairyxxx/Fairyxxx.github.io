# 三角函数关系公式

# (一) 倒数关系

  1. tan⁡xcot⁡x=1\tan{x}\cot{x}=1tanxcotx=1;
  2. sin⁡xcscx=1\sin{x}csc{x}=1sinxcscx=1;
  3. cos⁡xsecx=1\cos{x}sec{x}=1cosxsecx=1;

# (二) 商数关系

  1. tan⁡x=sinxcosx\tan{x}=\frac{sin{x}}{cos{x}}tanx=cosxsinx​;
  2. cot⁡x=cosxsinx\cot{x}=\frac{cos{x}}{sin{x}}cotx=sinxcosx​;

# (二) 平方关系

  1. sin⁡2x+cos⁡2x=1\sin^2{x}+\cos^2{x}=1sin2x+cos2x=1;
  2. 1+tan⁡2x=sec⁡2x1+\tan^2{x}=\sec^2{x}1+tan2x=sec2x;
  3. 1+cot⁡2x=csc⁡2x1+\cot^2{x}=\csc^2{x}1+cot2x=csc2x;

# 三角函数两角和差公式

  1. sin⁡(A+B)=sin⁡Acos⁡B+cos⁡Asin⁡B\sin{(A+B)}=\sin{A}\cos{B}+\cos{A}\sin{B}sin(A+B)=sinAcosB+cosAsinB;
  2. sin⁡(A−B)=sin⁡Acos⁡B−cos⁡Asin⁡B\sin{(A-B)}=\sin{A}\cos{B}-\cos{A}\sin{B}sin(A−B)=sinAcosB−cosAsinB;
  3. cos⁡(A+B)=cos⁡Acos⁡B−sin⁡Asin⁡B\cos{(A+B)}=\cos{A}\cos{B}-\sin{A}\sin{B}cos(A+B)=cosAcosB−sinAsinB;
  4. cos⁡(A−B)=cos⁡Acos⁡B+sin⁡Asin⁡B\cos{(A-B)}=\cos{A}\cos{B}+\sin{A}\sin{B}cos(A−B)=cosAcosB+sinAsinB;
  5. tan⁡(A+B)=(tan⁡A+tan⁡B)(1−tan⁡Atan⁡B)\tan{(A+B)}=\frac{(\tan{A}+\tan{B})}{(1-\tan{A}\tan{B})}tan(A+B)=(1−tanAtanB)(tanA+tanB)​;
  6. tan⁡(A−B)=(tan⁡A−tan⁡B)(1+tan⁡Atan⁡B)\tan{(A-B)}=\frac{(\tan{A}-\tan{B})}{(1+\tan{A}\tan{B})}tan(A−B)=(1+tanAtanB)(tanA−tanB)​;

# 三角函数积化和差公式

  1. sin⁡Asin⁡B=−cos⁡(A+B)−cos⁡(A−B)2\sin{A}\sin{B}=-\frac{\cos{(A+B)}-\cos{(A-B)}}{2}sinAsinB=−2cos(A+B)−cos(A−B)​;
  2. cos⁡Acos⁡B=cos⁡(A+B)+cos⁡(A−B)2\cos{A}\cos{B}=\frac{\cos(A+B)+\cos(A-B)}{2}cosAcosB=2cos(A+B)+cos(A−B)​;
  3. sin⁡Acos⁡B=sin⁡(A+B)+sin⁡(A−B)2\sin{A}\cos{B}=\frac{\sin(A+B)+\sin{(A-B)}}{2}sinAcosB=2sin(A+B)+sin(A−B)​;
  4. cos⁡Asin⁡B=sin⁡(A+B)−sin⁡(A−B)2\cos{A}\sin{B}=\frac{\sin(A+B)-\sin{(A-B)}}{2}cosAsinB=2sin(A+B)−sin(A−B)​;

# 三角函数和差化积公式

  1. sin⁡A+sin⁡B=2sin⁡(A+B)2cos⁡(A−B)2\sin{A}+\sin{B}=2\sin{\frac{(A+B)}{2}}\cos{\frac{(A-B)}{2}}sinA+sinB=2sin2(A+B)​cos2(A−B)​;
  2. sin⁡A−sin⁡B=2cos⁡(A+B)2sin⁡(A−B)2\sin{A}-\sin{B}=2\cos{\frac{(A+B)}{2}}\sin{\frac{(A-B)}{2}}sinA−sinB=2cos2(A+B)​sin2(A−B)​;
  3. cos⁡A+cos⁡B=2cos⁡(A+B)2cos⁡(A−B)2\cos{A}+\cos{B}=2\cos{\frac{(A+B)}{2}}\cos{\frac{(A-B)}{2}}cosA+cosB=2cos2(A+B)​cos2(A−B)​;
  4. cos⁡A−cos⁡B=−2sin⁡(A+B)2sin⁡(A−B)2\cos{A}-\cos{B}=-2\sin{\frac{(A+B)}{2}}\sin{\frac{(A-B)}{2}}cosA−cosB=−2sin2(A+B)​sin2(A−B)​;
  5. tan⁡A+tan⁡B=sin⁡(A+B)cos⁡Acos⁡B=tan⁡(A+B)1−tan⁡Atan⁡B\tan{A}+\tan{B}=\frac{\sin{(A+B)}}{\cos{A}\cos{B}}=\frac{\tan{(A+B)}}{1-\tan{A}\tan{B}}tanA+tanB=cosAcosBsin(A+B)​=1−tanAtanBtan(A+B)​;
  6. tan⁡A−tan⁡B=sin⁡(A−B)cos⁡Acos⁡B=tan⁡(A−B)1+tan⁡Atan⁡B\tan{A}-\tan{B}=\frac{\sin{(A-B)}}{\cos{A}\cos{B}}=\frac{\tan{(A-B)}}{1+\tan{A}\tan{B}}tanA−tanB=cosAcosBsin(A−B)​=1+tanAtanBtan(A−B)​;
